# Financial Insights in Banking Data  
**Power BI Analytics Project**

## Project Overview
This project is a comprehensive banking analytics case study developed using datasets provided by FinInsight Group, a consultancy specializing in data-driven banking insights. The analysis focuses on uncovering patterns in transaction behavior, customer financial profiles, and branch performance to support strategic decision-making in the banking sector.

Using Power BI, this project demonstrates end-to-end analytics capabilities, including data cleaning, data modeling, advanced DAX calculations, and interactive dashboard development. The final dashboard provides a holistic view of banking operations, customer behavior, and financial risk indicators.

---

## Objective
The objective of this project is to analyze banking transaction and customer account data to generate actionable insights that help financial institutions:

- Understand customer transaction behavior  
- Assess financial health and credit risk  
- Evaluate branch and account performance  
- Optimize banking products and services  
- Support risk management and strategic planning  

---

## Data Sources
### Dataset 1: Banking Transactions
**BankingDataset1.xlsx**

Includes detailed transaction-level information:
- Transaction ID  
- Account Number  
- Transaction Type  
- Transaction Amount  
- Transaction Date and Time  
- Branch Code  
- Currency  

### Dataset 2: Customer Account Details
**BankingDataset2.xlsx**

Contains account and customer-level attributes:
- Account Number  
- Account Holder  
- Account Type  
- Account Balance  
- Interest Rate  
- Credit Score  
- Opening Date  
- Loan Amount  
- Account Holder Demographics (employment sector, residence details, city)

## Key Analysis Performed
### Data Preparation and Modeling
- Imported and validated multiple datasets  
- Cleaned missing, duplicate, and inconsistent data  
- Created relationships using account numbers  
- Standardized currencies for comparison  

### Analytical Insights
- Average balances by account type  
- Branch-level transaction activity and performance  
- Transaction trends and seasonal patterns  
- Correlation analysis:
  - Interest rate versus account balance  
  - Credit score versus loan amount  
- Customer tenure and loyalty analysis  
- High-value and unusual transaction detection  
- Time-of-day transaction behavior patterns  

### Advanced Analytics
- Risk assessment model using DAX  
- Branch performance scoring based on volume and value  
- Customer demographic segmentation  
- Predictive modeling for account growth  
- Time-series forecasting of transaction volumes
